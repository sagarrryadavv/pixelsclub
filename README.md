## About
Pixels is BUIT's student coding club, a community for development, design, and innovation. We aim to **empower students** to learn, build projects, and collaborate on tech initiatives.

---

## Features
- **Workshops:** Hands-on sessions on Web Dev, AI/ML, Blockchain.  
- **PixelHacks:** Annual hackathon for innovation and learning.  
- **Club Projects:** Collaborate on real-world projects.
